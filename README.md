# Altiora Exim Website

Static website for Altiora Exim with gallery page, testimonials, and backend-ready contact form.

## Local development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the site server:
   ```bash
   npm start
   ```
3. Open `http://localhost:3000` in your browser.

## Features

- Homepage with products, services, testimonials, and contact form.
- Dedicated `gallery.html` page for product collections.
- Backend-ready `/api/contact` endpoint in `server.js`.
- Google Translate widget for international language support.
